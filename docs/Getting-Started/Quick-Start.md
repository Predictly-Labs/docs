# Quick Start

Get started with Predictly in 5 minutes.

## Requirements

- A wallet (Nightly, Petra, or Martian)
- MOVE tokens for voting

## Step 1: Connect Wallet

1. Open Predictly application
2. Click "Connect Wallet"
3. Select your wallet
4. Approve connection

## Step 2: Get MOVE Tokens

For testnet:

1. Copy your wallet address
2. Visit [Movement Faucet](https://faucet.movementnetwork.xyz/)
3. Request tokens
4. Wait for confirmation

## Step 3: Browse Markets

1. Go to Markets page
2. Browse active markets
3. Read market description
4. Check deadline and pool size

## Step 4: Place a Vote

1. Select a market
2. Click "Vote YES" or "Vote NO"
3. Enter stake amount
4. Confirm transaction
5. Approve in wallet

## Step 5: Claim Rewards

If you won:

1. Go to "My Votes"
2. Find resolved market
3. Click "Claim Reward"
4. Approve transaction
5. Receive MOVE tokens
