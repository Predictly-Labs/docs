# Wallet Setup

Set up your wallet to use Predictly.

## Supported Wallets

| Wallet  | Download                                          |
| ------- | ------------------------------------------------- |
| Nightly | [wallet.nightly.app](https://wallet.nightly.app/) |
| Petra   | [petra.app](https://petra.app/)                   |
| Martian | [martianwallet.xyz](https://martianwallet.xyz/)   |

## Installation

### Nightly Wallet

1. Visit [wallet.nightly.app](https://wallet.nightly.app/)
2. Download browser extension
3. Create new wallet
4. Save recovery phrase securely
5. Set password

### Petra Wallet

1. Visit [petra.app](https://petra.app/)
2. Add browser extension
3. Create new wallet
4. Backup recovery phrase
5. Complete setup

### Martian Wallet

1. Visit [martianwallet.xyz](https://martianwallet.xyz/)
2. Install extension
3. Create wallet
4. Store recovery phrase safely

## Connecting to Predictly

1. Open Predictly
2. Click "Connect Wallet"
3. Select your wallet
4. Approve connection request

## Getting MOVE Tokens

### Testnet

1. Copy wallet address
2. Visit [Movement Faucet](https://faucet.movementnetwork.xyz/)
3. Paste address
4. Click "Request Tokens"
5. Wait 10-30 seconds

Limit: 10 MOVE per 24 hours

## Security Best Practices

**Do:**

- Store recovery phrase offline
- Use strong password
- Verify URLs before connecting

**Don't:**

- Share recovery phrase
- Screenshot seed words
- Connect to unknown sites
