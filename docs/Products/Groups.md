# Groups

## Overview

Groups are private prediction communities where friends compete.

## Features

| Feature         | Description                 |
| --------------- | --------------------------- |
| Private markets | Group-exclusive predictions |
| Invite codes    | Control membership          |
| Leaderboards    | Track member performance    |
| Judge system    | Fair resolution             |

## Roles

| Role      | Permissions                                 |
| --------- | ------------------------------------------- |
| Admin     | Full control, manage members, assign judges |
| Judge     | Resolve markets                             |
| Moderator | Help manage content                         |
| Member    | Vote and create markets                     |

## Creating a Group

1. Click "Create Group"
2. Enter name and description
3. Upload image (optional)
4. Share invite code with friends

## Joining a Group

1. Get invite code from admin
2. Click "Join Group"
3. Enter code
4. Start participating
