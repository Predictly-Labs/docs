# Prediction Markets

## Overview

Predictly offers different types of prediction markets.

## Market Types

### Full Degen Markets

Standard prediction markets with full risk and reward.

| Aspect   | Details                     |
| -------- | --------------------------- |
| Risk     | Full stake at risk          |
| Reward   | Winners split losing pool   |
| Best for | High conviction predictions |

### Zero Loss Markets

Protected markets using DeFi yield farming.

| Aspect | Details             |
| ------ | ------------------- |
| Risk   | Principal protected |
| Reward | Winners earn yield  |
| Status | Coming soon         |

### Private Markets

Group-exclusive prediction markets.

| Aspect   | Details             |
| -------- | ------------------- |
| Access   | Group members only  |
| Topics   | Custom group topics |
| Best for | Friend competitions |

## How Markets Work

| Step | Action          | Who          |
| ---- | --------------- | ------------ |
| 1    | Create market   | Any user     |
| 2    | Vote YES/NO     | Participants |
| 3    | Deadline passes | System       |
| 4    | Resolve outcome | Judge        |
| 5    | Claim rewards   | Winners      |

## Market Parameters

| Parameter   | Description              |
| ----------- | ------------------------ |
| Title       | Market question          |
| Description | Detailed criteria        |
| Deadline    | When voting closes       |
| Group       | Public or specific group |
