# Overview

## What is Predictly?

Predictly is a **decentralized prediction market platform** built on Movement Network that enables users to create, participate in, and earn from prediction markets with friends and communities.

Unlike traditional prediction markets, Predictly focuses on **social prediction markets** where groups of friends can:

- Create custom prediction markets on any topic
- Compete with each other's forecasting skills
- Earn rewards for accurate predictions
- Build reputation as skilled predictors

## How Does It Work?

Predictly uses a **hybrid on-chain/off-chain architecture** to provide the best user experience:

1. **Create Markets** (Off-chain, FREE)
   - No gas fees for market creation
   - Stored in database for fast access
2. **Initialize Markets** (On-chain, Backend pays)
   - Backend deploys market to blockchain
   - Users don't pay gas for initialization
3. **Vote & Interact** (On-chain, Users pay)

   - Users vote with their wallets
   - Direct blockchain interaction
   - Transparent and verifiable

4. **Resolve & Claim** (On-chain)
   - Judges resolve markets
   - Winners claim rewards
   - All on blockchain
