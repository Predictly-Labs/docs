# Key Features

## Prediction Markets

Create and participate in YES/NO prediction markets on any topic.

| Feature        | Description                     |
| -------------- | ------------------------------- |
| Binary voting  | Vote YES or NO with MOVE tokens |
| Custom markets | Create markets on any topic     |
| Deadline-based | Voting closes at set time       |
| Transparent    | Results recorded on blockchain  |

## Market Types

### Full Degen Markets

Standard prediction markets with full risk/reward. Winners take the losing pool.

### Zero Loss Markets

Protected markets using DeFi yield farming. Principal protected, winners earn yield. (Coming soon)

### Private Markets

Group-exclusive markets. Only members can participate.

## Groups and Communities

| Feature       | Description                     |
| ------------- | ------------------------------- |
| Invite codes  | Control who joins               |
| Role system   | Admin, Judge, Moderator, Member |
| Group markets | Group-exclusive predictions     |
| Leaderboards  | Track member performance        |

## Judge System

Fair market resolution through trusted judges.

- Judges assigned by group admins
- Resolution recorded on blockchain
- Multiple judges supported per group
