# Welcome to Predictly

Predictly is a decentralized prediction market platform for friend groups built on Movement Network.

## What is Predictly?

Create, participate in, and earn from prediction markets with friends and communities. Stake MOVE tokens on your predictions and earn rewards for accurate forecasts.

| Feature            | Description                     |
| ------------------ | ------------------------------- |
| Prediction Markets | YES/NO markets on any topic     |
| Groups             | Private prediction communities  |
| Free Creation      | No gas fees to create markets   |
| Multi-wallet       | Nightly, Petra, Martian support |
| Fair Resolution    | Trusted judge system            |
